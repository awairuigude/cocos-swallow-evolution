# 吞食进化

基于 Cocos Creator 3.8.8 制作的 2D 吞噬成长游戏原型。玩家通过喷射移动、吞噬小型单位、躲避大型单位，在限时局内不断成长。

## 项目内容

- 喷射反作用力移动
- 质量消耗、成长与体型同步
- 食物、普通球、成长球、追逐球、巨型球
- 球体逃跑、追逐、吞噬与简单生态行为
- 危险轰炸区与资源期
- 局内三选一升级
- 结算统计与局内反馈

## 打开方式

1. 使用 Cocos Creator 3.8.8。
2. 打开本项目文件夹。
3. 运行 `assets/scenes/Main.scene`。
