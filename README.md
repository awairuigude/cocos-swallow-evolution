# 吞食进化

基于 Cocos Creator 3.8.8 制作的 2D 吞噬成长游戏原型。项目以“体型、质量、风险、收益”为核心，玩家通过喷射移动寻找食物和猎物，同时避开更大的球体、追逐球和危险轰炸区，在一局游戏内不断成长并选择强化方向。

## 基本信息

- 项目类型：2D 休闲生存 / 吞噬成长 / 局内强化原型
- 开发引擎：Cocos Creator 3.8.8
- 主场景：`assets/scenes/Main.scene`
- 主要语言：TypeScript
- 操作方式：按住鼠标进行喷射，球体会向喷射方向的反方向移动
- 单局时长：当前设计为较长局内成长节奏，可继续调整为 3 分钟、5 分钟或无尽模式

## 核心玩法

玩家控制一个可成长球体。球体的质量决定体型，质量越高，视觉尺寸越大，也更容易吞噬较小目标。玩家需要在地图中寻找食物和小型球体发育，同时避开远大于自己的巨型球和会索敌的追逐球。

移动本身不是无成本的。喷射会消耗质量，连续乱喷会触发更高消耗和恢复惩罚，因此玩家不能只靠一直高速移动解决问题。游戏希望形成一种明确的收支关系：赶路、追击和逃生都有成本，玩家需要判断一次行动能不能通过食物、猎物或资源期回本。

## 主要机制

### 质量与体型

- 玩家和敌人的体型都会随质量变化。
- 吞噬食物、吸收敌人、完整吃掉敌人都会增加质量。
- 喷射、被大球吞噬、进入危险区会减少质量。
- 质量变化会带来体型变化和视觉反馈，用于强化成长、受伤和吞噬感。

### 喷射移动

- 玩家按住鼠标时向鼠标方向喷射，球体向反方向获得速度。
- 喷射会持续消耗质量。
- 连续喷射会累积疲劳，超过一定比例后消耗显著提高。
- 低于惩罚阈值后进入惩罚恢复阶段，暂时限制行动，再进入补偿恢复阶段。

### 食物系统

- 地图中存在大量小型普通食物，用于基础发育。
- 资源期会生成更高收益的奖励食物。
- 食物被吃掉后会补充生成，保持局内资源流动。
- 小食物主要服务于早期发育，中后期玩家需要转向更高收益的吞噬目标。

### 敌人生态

游戏中不只有一种敌人，而是由多类球体组成一个简单生态：

- 小型猎物：主要作为玩家早期发育目标，移动慢，威胁低。
- 中型球体：会在地图中游走并吃食物成长，但不会主动追杀玩家。
- 成长球：更积极寻找食物，机动性更强，具有更明显的发育倾向。
- 追逐球：在一定条件下索敌追击玩家，但玩家接近或超过其质量后会转为逃跑。
- 巨型球：远大于玩家，移动慢但压迫感强，用来制造夹缝生存和路线选择压力。

### 吞噬与逃跑

- 只有质量明显占优的一方才能稳定吞噬另一方。
- 被吞噬的一方不会立刻消失，而是持续损失质量。
- 受损到一定阶段后，部分球体会尝试逃跑。
- 成长球、追逐球、巨型球在被吞噬到特定比例时，有概率获得逃脱机会。
- 球体损失越多，逃跑能力会发生变化；过小后会逐渐失去逃跑能力，避免被追上后长时间无法完整吃掉。

### 危险期与资源期

局内会周期性出现危险轰炸区和资源区组合：

1. 预警阶段：地图上出现红色范围提示，提醒玩家和敌人远离。
2. 危险阶段：区域内球体持续损失质量，大型球受到更明显影响，用于清空区域。
3. 资源阶段：部分危险区转化为黄色资源区，生成高收益食物，吸引玩家和其他球体争抢。
4. 冷却阶段：进入下一轮地图压力前的平稳期。

这个系统的作用不是单纯奖励玩家，而是制造风险和选择：危险区会改变局部生态，资源期会让玩家决定是否冒险抢发育。

### 局内强化

游戏加入了三选一局内升级，用于向肉鸽方向扩展。当前强化方向包括：

- 机动：提高喷射推力、最高速度、降低喷射惩罚。
- 猎杀：提高吞噬速度、完整吃掉球体后的奖励、外圈咬合能力。
- 资源：提高普通食物和资源食物收益，提前判断资源区。
- 生存：降低受伤和危险区损失，提高自然恢复能力。

这些升级让每局可以形成不同策略，例如偏向追猎、偏向资源争夺、偏向高机动逃生或偏向危险区生存。

## 项目亮点

- 不是简单的“吃小变大”，而是加入了喷射成本、疲劳惩罚、恢复补偿和资源回本判断。
- 敌人具有基础行为差异，不同球体在地图中会发育、逃跑、追逐或制造路线压力。
- 危险期和资源期把地图从静态空间变成动态局势，玩家需要观察时机和位置。
- 局内升级系统为后续肉鸽化留出了明确扩展方向。
- 结算统计会记录食物收益、吞噬收益、恢复收益、喷射消耗、敌人伤害和危险区伤害，方便后续继续调数值。
- 项目使用预制体、组件脚本、动态生成、碰撞判定、UI 状态、反馈特效等内容，覆盖了 Cocos 2D 项目的主要基础能力。

## 后续规划

- 继续调整成长速度、吞噬速度、喷射消耗和惩罚阈值，让“吃小球发育”和“乱喷赶路”形成更合理的收益差距。
- 优化红球、成长球、追逐球的逃跑表现，让不同类型球体的行为更容易区分。
- 增加更多局内升级，形成更明确的肉鸽构筑路线。
- 增加更多事件型地图机制，例如资源潮、危险边界、临时安全区、特殊猎物刷新。
- 后续如果时间允许，可以扩展为服务器多人联机玩法，让玩家之间也适用吞噬、逃跑、资源争夺和局内成长机制。

## 打开方式

1. 安装并打开 Cocos Creator 3.8.8。
2. 在 Cocos Dashboard 中选择本项目文件夹。
3. 打开 `assets/scenes/Main.scene`。
4. 点击运行预览。

## 目录说明

- `assets/scenes/`：场景文件
- `assets/scripts/`：核心 TypeScript 脚本
- `assets/prefabs/`：玩家反馈、食物、敌人等预制体
- `assets/materials/textures/`：项目使用的圆形贴图等资源
- `settings/`：Cocos Creator 项目配置
